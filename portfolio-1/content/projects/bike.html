ASL Translation Model

A deep learning model that translates American Sign Language (ASL) to text in real-time using LSTM neural networks. Built with PyTorch and OpenCV for hand gesture recognition and sequence modeling.

Key Features

  • Real-time ASL gesture recognition
  • LSTM-based sequence modeling
  • Custom data preprocessing pipeline
  • Webcam integration for live translation
  • 95% accuracy on common ASL phrases
  • Support for continuous signing

Technologies Used

PyTorch OpenCV Python LSTM Networks Computer Vision Deep Learning
