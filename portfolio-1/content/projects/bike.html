ASL Translation Tool

TranslateASL is an application for translating American Sign Language (ASL) into text using computer vision and machine learning.

1. Project Overview

The project features two distinct models: one for recognizing individual ASL letters and another for recognizing words and phrases from video sequences.

2. Methodology

Letter Model: Uses MediaPipe Hands for landmark extraction and an XGBoost classifier. It captures webcam frames, extracts hand landmarks in real time, and accumulates predicted letters into text.

Word Model: Uses MediaPipe Holistic for extracting pose, face, and hand landmarks. Sequences of frames are fed into an LSTM neural network (trained on video sequences) to classify actions.

3. Results & Outcome

The system successfully translates both static gestures (letters) and dynamic signs (words/phrases) in real-time using a standard webcam.

Key Features

  • Real-time ASL translation
  • Letter Recognition (XGBoost)
  • Word/Phrase Recognition (LSTM)
  • MediaPipe Landmark Extraction
  • Webcam Integration

Technologies Used

Python PyTorch OpenCV MediaPipe XGBoost LSTM
