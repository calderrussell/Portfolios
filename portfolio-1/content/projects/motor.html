3D Designed and Printed Motor

I made a motor that I fully 3D modeled and 3D printed, and wound the stator myself designing the WYE magnet design.

1. Project Overview

This project involved creating a functional electric motor from scratch using 3D printing technologies. I handled everything from the mechanical design in Fusion 360 to the electrical winding of the stator.

2. Objective

I wanted to build a motor that I would potentially use in a future drone build.

3. Methodology

I used Fusion 360 to model it and started by laying out the goals for KV and torque, letting that influence the choices I made when it came to winding and layout. Choosing to use an ESC also changed my design decisions.

4. Results & Outcome

I was able to make it work; it runs and can be used. I haven't added a prop to it yet to test it for flight, but the core mechanism is functional.

5. Next Steps

I would like to machine one in the future instead of using 3D printed parts so that it is stronger and improves efficiency. I also want to try making an axial motor.

Key Features

  • Custom 3D design tailored for optimal performance
  • Self-wound stator for improved efficiency
  • KV measurement for precise speed control
  • Testing capabilities for various RPMs
  • Compact and lightweight design

Technologies Used

Fusion 360 E&M 3D Printing Winding
